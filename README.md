# ontwerpkeuze UI

Mijn ontwerpkeuze heb ik gebaseerd op mijn eigen voorkeuren.
ik wil vooral dat animaties niet te druk of te veel zijn, daarom heb ik de bewegingen redelijk 'basic' gehouden. 
Ook heb ik ervoor gezorgd dat animaties niet te lang duren omdat ik ook vind dat dat tijdrovend is.
Ik heb dus op basis van de hints mijn eigen ideeën geimplementeert en vind het resultaat persoonlijk erg goed.
